## Instalación #

```bash
$ git clone https://github.com/YeisonGomez/scraping-chaira-nodejs.git
$ cd scraping-chaira-nodejs
$ npm i
$ npm start
...
Servidor corriendo en 127.0.0.1:8099
```

Abrir el navegador web y acceder a la siguiente dirección:

 - http://localhost:8099/user/login?user=p.prueba&password=123

Donde **p.prueba** es el nombre de usuario, y **123** es la contraseña correspondiente a la cuenta de [Chaira](https://chaira.udla.edu.co/Chaira/Logon.aspx)
